Detailed Description

This file contains definitions for coveo::lazy::map and coveo::lazy::multimap. These lazy-sorted containers mimic the standard C++ containers of the same name, but perform lazy sorting.

These definitions are just typedefs. For actual implementation, see coveo/lazy/detail/lazy_sorted_container.h.

template<class K , class T , template< class _AllocT > class _Alloc = std::allocator>
using coveo::lazy::map_allocator = typedef detail::map_allocator<K, T, _Alloc>

Default allocator type for lazy maps. Use this to specify a custom allocator for coveo::lazy::map or coveo::lazy::multimap:

// my_custom_allocator must be a template accepting a single type argument
coveo::lazy::map_allocator<key_type, mapped_type, my_custom_allocator>
Template Parameters
KType of keys stored in the map.
TType of values stored in the map.
_AllocBase allocator actually used to perform allocation. Must be a template type accepting a single argument. By default, the allocator type is std::allocator.
