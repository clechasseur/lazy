Detailed Description

template<class Container>
class coveo::lazy::insert_iterator< Container >

Equivalent of std::insert_iterator usable with lazy sorted containers. Always performs blind insertions. Mirrors http://en.cppreference.com/w/cpp/iterator/insert_iterator
